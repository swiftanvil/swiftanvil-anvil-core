# ``AnvilCore``

Shared infrastructure for the SwiftAnvil ecosystem.

## Overview

AnvilCore provides foundational types used across SwiftAnvil packages:

- ``AnvilLogger`` — Structured logging with severity levels
- ``AnvilConfiguration`` — Actor-isolated configuration store
- ``AnvilTask`` — Sendable concurrent task wrapper
